Funfetti-chicken

Music analysis software that uses sentiment analysis to decide what foods go with what music

Download this project as a .zip file Download this project as a tar.gz file

Welcome to Funfetti Chicken

This is a group project the four of us are working on for a class at Franklin W. Olin College of Engineering. The purpose of this project is to better understand how sentiment analysis works and to refine it to pick up more than just positive vs. negative emotions.
